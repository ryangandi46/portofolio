SITEPAD - Sistem Pendataan Aset Daerah

Tentang Proyek

SITEPAD adalah sebuah aplikasi web yang dikembangkan untuk membantu Badan Keuangan dan Aset Daerah (BKAD) Tangerang Selatan dalam mengelola data keuangan dan aset daerah. Proyek ini bertujuan untuk menggantikan sistem manual yang ada dengan solusi digital yang lebih efisien, transparan, dan terintegrasi.

Fitur Utama

  • Dashboard Interaktif: Menampilkan ringkasan data aset dan keuangan secara visual menggunakan grafik dan statistik.
  • Manajemen Aset (CRUD): Fitur lengkap untuk menambah, membaca, memperbarui, dan menghapus data aset daerah.
  • Manajemen Pengguna: Sistem hak akses (Role-Based Access Control) untuk membedakan antara admin dan pengguna biasa.
  • Fitur GIS (Leaflet JS): Visualisasi aset di atas peta interaktif untuk memudahkan pemantauan lokasi.
  • Laporan & Ekspor Data: Kemampuan untuk menghasilkan laporan dan mengekspor data ke format Excel atau PDF.

Teknologi yang Digunakan

Laravel 11 PHP 8.2 MySQL Tailwind CSS JavaScript (ES6+) Leaflet JS ApexCharts

Tantangan & Solusi

Tantangan: Mengintegrasikan data aset yang beragam dari berbagai sumber menjadi satu database yang terstruktur.
Solusi: Merancang skema database yang normal dan relasional dengan cermat, serta membuat proses migrasi data yang terencana.
