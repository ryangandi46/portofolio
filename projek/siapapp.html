siapApp - Fondasi Aplikasi Laravel (Boilerplate)

Tentang Proyek

siapApp adalah sebuah sistem inventaris aset sekolah dibuat menggunakan Laravel yang dirancang untuk mempercepat proses awal pencatatan aset disekolah yang dimana untuk menggantikan pencatatan manual. Proyek ini menyediakan modul-modul inti yang sudah terkonfigurasi, seperti manajemen pengguna, autentikasi, import & export data aset dan peminjam.

Fitur Utama

  • Sistem Autentikasi Lengkap: Fungsionalitas login, registrasi, dan proteksi rute (middleware) yang sudah terpasang.
  • Manajemen Hak Akses (Otorisasi): Pengaturan hak akses pengguna (role-based) yang fleksibel menggunakan Gates & Policies Laravel.
  • Pola CRUD Standar: Menyediakan contoh operasi Create, Read, Update, Delete (CRUD) yang generik, lengkap dengan validasi (FormRequest) sebagai acuan.
  • Laporan: Menyediakan laporan aset dan peminjaman dalam format PDF dan Excel untuk kemudahan dokumentasi.
  • Import Data: Menyediakan fitur impor data aset dari file Excel untuk mempercepat proses input data awal.
  • Komponen UI Terstruktur: Termasuk layout Blade utama dan komponen-komponen yang dapat digunakan kembali (seperti notifikasi/alert) untuk konsistensi tampilan.

Teknologi yang Digunakan

Laravel (PHP) Blade/Bootstrap MySQL Laravel Excel PHPUnit Git

Peran & Kontribusi

Peran utama saya adalah merancang dan membangun sistem invenraris ini dari awal. Kontribusi spesifik saya meliputi standarisasi pola CRUD, menyiapkan sistem agar tidak ada kendala dalam penggunaannya.

Dampak Proyek

Dengan adanya siapApp, proses pencatatan aset sekolah menjadi lebih efisien dan terstruktur. Proyek ini telah membantu sekolah dalam mempermudah pencatatan dan pelacakan aset, mengurangi kesalahan manusia, serta meningkatkan transparansi dalam pengelolaan inventaris.
